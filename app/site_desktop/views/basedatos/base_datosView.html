Contenido del paquete

La base de datos es un archivo comprimido con extensión .zip, el cual incluye los siguientes documentos:

  • GTO_GOB_2018.csv
  • GTO_DIP_LOC_2018.csv
  • GTO_AYUN_2018.csv
  • GTO_GOB_CANDIDATURAS_2018.csv
  • GTO_DIP_LOC_CANDIDATURAS_2018.csv
  • GTO_AYUN_CANDIDATURAS_2018.csv
  • LEEME.txt

Los siguientes archivos contienen datos por: Entidad, Distrito, Sección y Casilla de votos por partido político, candidato independiente, coalición, votos nulos y totales.

  • GTO_GOB_2018.csv
  • GTO_DIP_LOC_2018.csv
  • GTO_AYUN_2018.csv

El archivo:

GTO_GOB_CANDIDATURAS_2018.csv contiene la lista de candidatos a Gobernador, considerando datos de Entidad, Distrito, Partido y Candidato Independiente.

GTO_DIP_LOC_CANDIDATURAS_2018.csv contiene la lista de candidatos independientes y suplentes a Diputado Local, por Entidad, Distrito, Partido y Candidato Independiente.

GTO_AYUN_CANDIDATURAS_2018.csv contiene la lista de candidatos independientes y suplentes a Presidente Municipal, por Entidad, Distrito, Partido y Candidato Independiente

LEEME.txt presenta un resumen del contenido de los archivos para una referencia rápida.

Descarga

Para iniciar la descarga del paquete oprime el botón:

Descargar base de datos

Localiza el archivo en la ubicacion que hayas elegido, o bien en tu historial de descargas. Puedes hacerlo presionando:

CTRL + J en Windows

Alt + Cmd + L en Mac

Se abrirá la carpeta de Descargas, ubica el archivo y selecciónalo.

Descomprime el archivo

Haz clic derecho sobre el archivo y selecciona:

Extraer aquí para guardar en la misma carpeta
Extraer ficheros... si deseas guardar en otra ubicación

Una vez realizada esta acción, se abre una ventana mostrando el proceso de extracción que se muestra en la imagen.

Al terminar, tendrás el archivo descomprimido.

Nota:
Al tratarse de un archivo comprimido, es necesario tener el software Winrar o 7-Zip instalado en el caso de Windows; o bien Zipeg, Expander o RAR para usuarios Mac. Si tienes instalado Winzip, no funcionará.

Abre los archivos

Haz doble clic sobre alguno de los archivos extraídos.

Recuerda, los archivos csv contienen información que puedes ver en programas de hoja de cálculo (como Excel, Access, Calc Open Office, Smartsheet, entre otros).

El archivo LEEME.txt abre con el bloc de notas en Windows o Notas en Mac.

Consulta

Finalmente, accedes a la información que contiene la base de datos obtenida a través de las Actas de Escrutinio y Cómputo que procesó el PREP.

Comprueba el paquete
